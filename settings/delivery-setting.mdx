## **Nhà cung cấp vận chuyển**

---

### **Tổng quan**

Truy cập **Quản lý Mini App → Nhà cung cấp vận chuyển**.\
Chọn đơn vị vận chuyển để kết nối.

> Hiện tại Aimini hỗ trợ đơn vị: **GHN (Giao Hàng Nhanh)**.

### **Giao Hàng Tiết Kiệm (GHTK)**

**Bước 1:**\
Truy cập https://khachhang.giaohangtietkiem.vn/web/dang-ky → Đăng nhập tài khoản GHTK.

<Frame>
  <img
    src="https://aimini-upload.sgp1.digitaloceanspaces.com/image624_a654842704.png"
    alt="Đăng nhập GHTK"
    style={{ borderRadius:"0.5rem" }}
  />
</Frame>

**Bước 2:**\
Sau khi đăng nhập, chọn **Quản trị → Thông tin tài khoản** để lấy **API Token Key**.

<Frame>
  <img
    src="https://aimini-upload.sgp1.digitaloceanspaces.com/image625_10b086307e.png"
    alt="Lấy API Key GHTK"
    style={{ borderRadius:"0.5rem" }}
  />
</Frame>

<Frame>
  <img
    src="https://aimini-upload.sgp1.digitaloceanspaces.com/image626_ba34fc2ace.png"
    alt="Copy API Key GHTK"
    style={{ borderRadius:"0.5rem" }}
  />
</Frame>

**Bước 3:**\
Truy cập **POS → Nhà cung cấp vận chuyển → Kết nối GHTK**.

<Frame>
  <img
    src="https://aimini-upload.sgp1.digitaloceanspaces.com/image694_7e8eefce99.png"
    alt="Kết nối GHTK trong POS"
    style={{ borderRadius:"0.5rem" }}
  />
</Frame>

**Bước 4:**\
Nhập **API Token Key** → Nhấn **“Tiếp theo”** để kết nối.

<Frame>
  <img
    src="https://aimini-upload.sgp1.digitaloceanspaces.com/image695_98d9880987.png"
    alt="Nhập API Key kết nối GHTK"
    style={{ borderRadius:"0.5rem" }}
  />
</Frame>

Kết nối thành công:

<Frame>
  <img
    src="https://aimini-upload.sgp1.digitaloceanspaces.com/image629_072cd11b2f.png"
    alt="Kết nối GHTK thành công"
    style={{ borderRadius:"0.5rem" }}
  />
</Frame>

> **Lưu ý:** Bạn cần có tài khoản GHTK để thực hiện kết nối.
